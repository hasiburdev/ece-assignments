# 03 - An array A Containing N elements is given. Write a program that captures the sum of array elements.

```c
#include<stdio.h>

int main() {
int n;
printf("Enter the length of the array: ");
scanf("%d", &n);

    int arr[n];
    printf("Enter the elements of the array: ");
    for(int i = 0; i < n; i++) {
        scanf("%d", &arr[i]);
    }

    int sum = 0;
    for(int i = 0; i < n; i++) {
        sum += arr[i];
    }
    printf("The sum of the array is: %d\n", sum);
}

```
